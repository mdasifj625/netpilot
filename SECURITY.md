# Security Policy

## Supported Versions

Currently, the `main` branch and the latest release tag are supported with security updates.

## Reporting a Vulnerability

We take the security of NetPilot seriously. If you discover a security vulnerability in this project, please report it immediately.

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, please send an email to **mdasifj625@users.noreply.github.com** with a description of the vulnerability. We will work with you to understand the issue and provide a fix as quickly as possible.

We will try to acknowledge your email within 48 hours.
